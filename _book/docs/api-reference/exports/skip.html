skip

Export skip

Type definition:

  • skip(name: string, implementation: Implementation): void
  • Implementation = (t: Test) => Promise<void>
  • Test

Example usage:

const {skip} = require('cybernaut');

// This end-to-end test won't be executed (and marked as SKIP)
skip('foo', async t => {
  // ...
});
